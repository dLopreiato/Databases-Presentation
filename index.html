What a Database Is

In essence, a database is a file with different interface.

File

  • Read, Write, Seek
  • Character/Line Granularity
  • No Pre-defined Structure

Database

  • Create, Read, Update, Delete
  • Datatype Granularity
  • Row Column Structure
